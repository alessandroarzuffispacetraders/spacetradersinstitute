# IST — Space Traders Institute

## Cos'è
Scuola di coaching 1:1 per trader. Programma da 90 giorni con coach dedicato, mental coach, videocorsi, community e sessioni Zoom.

## Stack tecnico
- Frontend: React + Vite + TypeScript
- Styling: Tailwind CSS
- Backend: Node.js + Express
- Database: PostgreSQL + Supabase
- Auth: Supabase Auth
- Real-time chat: Supabase Realtime
- Storage video: Cloudflare R2

## Ruoli utente
1. Studente
2. Coach
3. Mental Coach
4. Admin

## Area Studente
- Percorso guidato in 4 fasi: Onboarding → Build → Test → Deploy
- Videocorsi con stato completamento
- Diario di trading
- Progressi, badge, milestone
- Chat: community generale + gruppo privato con coach + DM
- Area mental coach con materiali e task
- Live e replay sessioni Zoom
- Link esterno al Protocol Data Journal

## Area Coach
- Lista studenti assegnati con progressi
- Review esercizi e commenti privati
- Segnalazione studenti bloccati

## Area Mental Coach
- Lista studenti assegnati
- Stato sessioni (sessione 1 e 2)
- Note private per studente
- Chat privata con studente

## Area Admin
- Gestione utenti e ruoli
- Gestione contenuti (videocorsi, live)
- Statistiche generali
- Stato studente (attivo/scaduto/bloccato)

## Regole importanti
- Nessun pagamento in-app
- L'app riceve solo lo stato di accesso dal backend (attivo/scaduto/bloccato)
- Protocol Data Journal è solo un link esterno, nessuna integrazione
- Design scuro, moderno, professionale per trader
- Navigazione sidebar su desktop, bottom nav su mobile
